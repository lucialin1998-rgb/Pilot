# Simple Static Website

This project is a basic 3-page static website built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` — Home page
- `projects.html` — Projects page with 3 placeholder cards
- `contact.html` — Contact page with a form and client-side submit alert

## How to run locally

1. Open a terminal in this folder.
2. Start a local server:

   ```bash
   python3 -m http.server 8000
   ```

3. Open your browser at:

   ```
   http://localhost:8000
   ```

## Notes

- No frameworks or build tools are required.
- The layout is responsive and optimized for desktop and mobile.
